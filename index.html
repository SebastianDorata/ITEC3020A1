About

Welcome to Game Store, a platform created for gamers to easily discover and explore popular video games across different platforms. Our goal is to provide a simple and organized space where users can browse games, view important details such as price, platform, and editions, and find recommended titles all in one place.

Game Store features a collection of trending and recommended games designed to help players quickly find games that match their interests. The website focuses on a clean design and smooth navigation to create an enjoyable browsing experience for users.

This website was developed as a web application project demonstrating modern front-end development concepts such as interactive navigation, dynamic game displays, and form validation. Game Store aims to make game discovery easy, engaging, and accessible for everyone.
